# demo-astro

Public Astro demo site with multiple premium-style example businesses.

## What is included

- 8 standalone example routes
- 1 gallery homepage
- reusable capsule-based sections
- local dummy media for safe public sharing

## Routes

- `/`
- `/northstar-grid/`
- `/elm-and-ivory/`
- `/harborcare-clinics/`
- `/fieldstone-living/`
- `/briar-and-stone-counsel/`
- `/brightpath-academy/`
- `/saffron-house-hotel/`
- `/aureline-objects/`

## Local development

Use Node 22.

```bash
npm install
npm run dev
```

## Validation

```bash
npm run check
npm run build
```

## Deployment

This project is static and can be deployed directly to Vercel or any static hosting provider.
